Piano on the Web

Hey there! This site allows you to make music like you would on any piano! Let's make some noise!

Press the S key to make a middle C (or C4), then D to make D4 and so on up until B4 with the K key.

Press the Shift key to make a sharp sound. For example, Press Shift and S to play a middle C#.

Use → to move up piano sections and use ← to move down piano sections. The current section is 4.
